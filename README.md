# 🧾 فاکتورساز آنلاین

یک وب اپلیکیشن مدرن، سریع و متن‌باز برای ساخت، پیش‌نمایش، اشتراک‌گذاری و دانلود فاکتورهای حرفه‌ای. این پروژه با استفاده از **Next.js** و **Tailwind CSS** توسعه داده شده و خروجی‌های PDF آن به صورت کامل با زبان فارسی سازگار است.

## ✨ قابلیت‌ها و ویژگی‌ها

- 🎨 **طراحی زیبا و واکنش‌گرا**
- 🖼️ **آپلود لوگو:** امکان آپلود لوگوی شخصی (PNG/SVG) و قرارگیری دقیق آن در فایل PDF.
- 🖌️ **شخصی‌سازی رنگ‌ها:** قابلیت انتخاب رنگ قالب فاکتور.
- 🧮 **محاسبات خودکار:** محاسبه دقیق جمع کل، مالیات (با درصد دلخواه) و تخفیف (به صورت درصدی یا مبلغ ثابت).
- 📝 **یادداشت‌های پویا:** امکان افزودن شرایط پرداخت و یادداشت‌های دلخواه با قابلیت تنظیم خودکار ارتفاع در PDF.
- 📄 **تولید PDF هوشمند:** ساخت فایل PDF با استفاده از [jsPDF](https://github.com/parallax/jspdf).
- 💾 **ذخیره خودکار پیش‌نویس:** تمام تغییرات به صورت آنی در مرورگر ذخیره می‌شوند تا اطلاعات شما با بستن صفحه از بین نرود.
- 🔗 **اشتراک‌گذاری آسان:** قابلیت تولید لینک اشتراک‌گذاری موقت برای ارسال سریع فاکتور به مشتری.

## 🛠️ تکنولوژی‌های استفاده شده

- **Framework:** [Next.js (App Router)](https://nextjs.org/)
- **UI:** [React 18](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **PDF Generator:** [jsPDF](https://parall.ax/products/jspdf)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 راه‌اندازی

1. مخزن را کلون کنید

```bash
git clone https://github.com/chaveamin/invoice-app.git
```

2. وارد پوشه پروژه شوید

```bash
cd invoice-app
```

3. پکیج‌های مورد نیاز را نصب کنید

```bash
npm install
```

4. سرور را اجرا کنید

```bash
npm run dev
```

5. مرورگر خود را باز کرده و به آدرس `http://localhost:3000` بروید.
